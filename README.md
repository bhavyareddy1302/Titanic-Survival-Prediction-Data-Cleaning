# Titanic Survival Prediction – Data Cleaning Project

## Overview
This project demonstrates data preprocessing on the Titanic dataset using Python.

## Technologies Used
- Python
- Pandas
- Matplotlib
- Scikit-learn

## Tasks Performed
- Data exploration
- Handling missing values
- Label Encoding
- One-Hot Encoding
- Data visualization
- Exporting cleaned dataset

## Output
The cleaned dataset is saved as `Titanic_Cleaned.csv`.

## Conclusion
This project prepares the Titanic dataset for machine learning applications.
